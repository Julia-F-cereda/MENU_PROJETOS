<h3>DESCRIÇÃO DO PROJETO</h3>
Este projeto foi desenvolvido com o objetivo de armazenar e organizar trabalhos realizados, funcionando como um portfólio digital. Ele reúne projetos e atividades desenvolvidas ao longo dos estudos, facilitando a visualização e apresentação das produções.

<P>2. Abra o arquivo `index.html`.</P>
<P>3. Execute utilizando a extensão Live Server no VS Code.</P>
